# TV vendors
Find similar TV vendors based on remote codes

## Usage:
$ python3 remote.py -h
usage: remote.py [-h] -v YOUR_VENDOR [-m MATCHES]

optional arguments:
  -h, --help            show this help message and exit
  -v YOUR_VENDOR, --your_vendor
                        Your TV vendor name
  -m MATCHES, --matches
                        Top matches count
